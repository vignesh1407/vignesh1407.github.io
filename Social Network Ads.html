Problem Statement¶

In this Use Case we will be working with Social Network Ads data set, we will be dealing with all the user details that are given in the time of registering.

With this data, we will try to create a model that will predict, whether the new user will buy the new SUV or not, based on the details of the user given at the time of registeration.

We will be using all types of classification models and we choose which model provide us high accuracy.

Models:

  1. Logistic Regression
  2. Decision Tree
  3. Random Forest
  4. KNN Classifier

Result¶

The Accuracy Levels of the Models is observed to be:

1.Logistic Regression Classifier - 44%
2.Decision Tree Classifier - 88%
3.Random Forest Classifier - 91%
4.KNN Classifier - 98%

Out of the four, KNN Classifier gives out 98% Accuracy. So, For the Social Network Ads Model, we can suggest K Nearest Neighbor Algorithm gives the Best accuracy.
